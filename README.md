# 🎾 Zarautz-ChatBot: Asistente Virtual Bilingüe

Asistente inteligente para el **Club Deportivo Zarautz** (Zarautz Kirol Elkartea) integrado con la API oficial de WhatsApp Cloud.

## 🚀 Características
- **Inteligencia Artificial:** Potenciado por **Llama 3.3 70B** a través de **Groq** para una velocidad extrema.
- **Bilingüismo Real:** Respuestas automáticas y fluidas en **Castellano** y **Euskera Batua**.
- **Memoria de Conversación:** Capacidad para recordar el contexto de la charla (nombres, dudas anteriores, etc.).
- **Información del Club:** Gestión de horarios, tarifas de pádel y contacto directo.

## 🛠️ Stack Tecnológico
- **Backend:** Node.js + Express
- **API IA:** Groq (Llama 3.3)
- **Mensajería:** Meta WhatsApp Business API
- **Herramientas:** Axios, Dotenv, Ngrok

## 📋 Requisitos Previos
1. Node.js instalado.
2. Cuenta en [Meta for Developers](https://developers.facebook.com/).
3. API Key de [Groq Console](https://console.groq.com/).
